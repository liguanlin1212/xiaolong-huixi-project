# Demo Explain

Version: 1.0.0

## Overview
This demo directory layout provides placeholders for the narrative classification workflow.

## Data Samples
- `data/raw_texts.json` contains time-stamped inputs to be analysed.
- `data/results.json` contains the model outputs for downstream display.

## Service Flow
1. The API layer receives payloads for analysis.
2. The narrative service orchestrates judgement extraction.
3. The UI layer renders a timeline view of the resulting nodes.
