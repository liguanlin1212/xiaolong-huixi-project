# 技术背景文档

## 一、比赛总体技术定位

### 1️⃣ 平台属性
- **平台**：骁龙 X Elite AIPC
- **系统**：Windows 11 ARM
- **类型**：PC 端 AI 应用
- **运行环境**：真实 AI PC（非云服务器）

### 2️⃣ 开发模式
- **采用**：本地开发 + 远程 AIPC 验证
- **具体含义**：
  - 你在个人电脑上开发
  - 在分配时间段远程连接 AIPC 真机
- **在 AIPC 上**：
  - 部署模型
  - 验证 NPU 调用
  - 测试性能
  - 运行最终 Demo

⚠ 数据不会保留，需自行备份。

## 二、硬件与算力结构

AIPC 具备异构算力架构：
- **CPU** —— 通用计算
- **GPU** —— 并行计算 / 图形
- **NPU** —— AI 专用低功耗推理单元

比赛重点考察：
- 是否真实使用 NPU
- 是否合理设计异构计算
- 是否清晰说明算力分工

这意味着：

## 四、模型层面的现实情况

### 1️⃣ 大模型
- **如**：Qwen 7B / 14B
- **特征**：
  - 生成式
  - 参数量大
  - 占用内存高
  - 推理成本高
- **适合**：
  - 解释
  - Agent 决策
  - 文本生成

### 2️⃣ 轻量模型
- **如**：
  - 文本分类
  - NLI（文本蕴含）
  - 小型 Transformer
  - 向量模型
- **特征**：
  - 小参数量
  - 输出结构化
  - 推理快
  - 易量化
  - 非生成式
- **适合**：
  - 跑在 NPU 上
  - 实时判断任务
